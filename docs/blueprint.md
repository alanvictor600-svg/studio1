## Core Features:

- Ticket Selection: Allows the user to pick a set of 10 numbers between 1 and 25, allowing the repeat of up to 4 times.
- Ticket Management: Displays active, expired and winning tickets
- Auto-Fill Feature: Simulates ticket auto-generation with number repetition according to the app's logic, making the choice faster

## Style Guidelines:

- Dominant: Forest Green (#228B22), which inspires a sense of trust.
- Secondary: Gold accents (#FFD700), representing wealth and success.
- Accent: Crimson (#DC143C) to indicate when a ticket has won a prize.
- Clear, legible fonts for easy number reading
- Custom-designed icons, particularly for filtering or showing different status of each ticket
- Grid-based layout that clearly displays the status for each ticket, allowing quick scanning.
